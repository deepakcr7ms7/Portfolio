System Architecture Overview

A rules-driven pricing platform designed to eliminate manual cost calculation, ensure consistency across sites, and provide full auditability.

  • Step-based workflow engine enforces consistency and prevents invalid state changes
  • Pricing computed on demand using latest effective rates (no stale data persisted)
  • End-to-end audit trail for compliance, reviews, and historical analysis

Key Design Decisions

Workflow Enforcement

Step-based statuses prevent unauthorized or accidental changes after approvals.

Unified Pricing Model

Replaced fragmented Excel formulas with a single authoritative rate table.

Dynamic Cost Calculation

Final cost calculated on-demand using latest effective rates.

System-Driven Logic

User inputs guide pricing, system enforces correctness.

Design Trade-offs & Learnings
  • Reduced flexibility to enforce consistency
  • Historical open jobs may reflect updated pricing
  • Some granular cost breakdowns removed for clarity
  • Strong backend validation required due to reduced user control
